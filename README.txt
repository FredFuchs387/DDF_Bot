Moderator Commands:

!mediashareon/!mediashareoff
	Default: OFF
	Toggles mediashare message in chat 
	
!nukeon/!nukeoff
	Default: OFF
	Toggles chat nuke 
		-Sets slow mode to where messages can only be sent every 30s
		-Non-Moderator/VIP users are timed out for 30s for any message
		-Sets follower duration to 3 days
		-Sets chat to subscribers only
	
!russianon/!russianoff
	Default: OFF
	Toggles allowability for cyrillic charset
